# 🐾 PetStoreAPI

PetStoreAPI es una API REST desarrollada para la gestión de una tienda de mascotas. Permite administrar información de mascotas, usuarios y órdenes de compra, facilitando operaciones CRUD mediante servicios web.

---

## 🚀 Tecnologías utilizadas

- Java
- Spring Boot (si aplica, ajusta si usas otro framework)
- Maven / Gradle
- REST API
- JSON
- Git & GitHub

---

## 📌 Características

- Registro de mascotas
- Consulta de mascotas por ID o lista general
- Actualización de información de mascotas
- Eliminación de registros
- Gestión básica de usuarios (si aplica)
- Arquitectura RESTful
